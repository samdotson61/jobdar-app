# modes/es — Spanish modes (full parity)

This directory holds the **full Spanish** peer of every base mode in `modes/`. Spanish is a
first-class language, not a partial machine translation.

> **Phase 0 scaffold.** Authoring the Spanish modes is **Phase 1.2** (i18n layout) / **1.5**.
> Until then, the agent falls back to the canonical English modes one level up.
