# Shared rubric (canonical, American English)

This file holds the scoring rubric and shared instructions that every mode builds on.

> **Phase 0 scaffold.** The full rubric — fit scoring, level-fit ranking (no penalty for a
> level the user selected), no-degree tuning, and region/comp tuning — is authored in
> **Phase 1.1** and refined in **Phase 4** (levels) and **Phase 5** (regions).

## Principles (locked now)

- **Entry-default, senior opt-in.** Rank any *selected* level on merit; only de-prioritize
  roles *above* the user's highest selected level.
- **No-degree path is first-class.** Treat "Bachelor's required" as a soft signal; surface
  "or equivalent experience", apprenticeships, and skills-based roles. Never silently hide
  degree-gated roles — flag them as a stretch.
- **Bilingual.** Generate output in the user's `language` (or the JD's language).
- **Local & honest.** Use only public job data + the user's local résumé; cite the JD.
